# Predicción de precios de vuelos

## Objetivo
Desarrollar un modelo de Machine Learning para predecir el precio de vuelos, utilizando variables como aerolínea, duración, escalas y fecha de viaje, con el fin de apoyar la toma de decisiones en la compra de pasajes.

## Dataset
Conjunto de datos de vuelos que incluye información sobre aerolínea, origen, destino, duración, número de escalas, fecha y precio.

## Metodología
- Definición del problema de negocio y variable objetivo (precio del vuelo)
- Análisis de calidad de datos
- Análisis exploratorio (EDA)
- Análisis de correlaciones para identificar variables influyentes
- Transformación y creación de variables categóricas y numéricas
- Construcción de función de procesamiento para obtener un dataset limpio
- Entrenamiento de modelos y optimización de hiperparámetros mediante grilla de búsqueda
- Evaluación de modelos utilizando múltiples métricas y comparación con baseline

## Resultados
- Identificación de variables relevantes para la predicción del precio
- Mejora del desempeño respecto a un modelo baseline
- Selección del mejor modelo en base a métricas de evaluación

## Tecnologías utilizadas
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
